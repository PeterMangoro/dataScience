# Peter Mangoro - ML Demos

An interactive portfolio of four machine learning projects, served as tabs:

1. **California Housing** - regression predicting median house value from census block-group features (engineered linear model).
2. **Pima Diabetes** - binary classification of diabetes risk from clinical measurements (balanced logistic regression).
3. **Online Retail** - neural sequence model (Transformer) recovering customer segments from purchase history, with attention diagnostics.
4. **Amazon Review Sentiment** - TF-IDF + linear classifier predicting positive vs negative review sentiment.

Built from coursework notebooks; see the source repository for full analysis and reports.
